Welcome to the Find ESSJ Taco Truck App! This application helps users find the nearest taco trucks and check their location. It's built with React Native and provides a user-friendly experience for taco lovers.

About the App:

The Taco Truck App is designed to make it easier for users to discover delicious taco options near them. With real-time location, users can quickly find taco trucks. Our goal is to connect taco lovers with the best local food trucks and enhance their dining experience.

Features:

  • Real-time location of taco trucks

Technologies Used:

This app is developed using:

  • React Native for mobile app development
  • Google Maps API for location services
